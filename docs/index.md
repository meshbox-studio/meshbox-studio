# Meshbox Studio Docs

Meshbox Studio is a lightweight, self-hosted workspace for 3D printing enthusiasts to archive projects, store print files, and keep practical notes tied to real outcomes.

!!! warning "Early development"
    This documentation is currently a placeholder while the core product architecture is being built.

## What this project is for

- Keep slicer profiles, models, and notes in one place
- Track print iterations and document failures or fixes
- Build a personal archive that stays searchable over time

## Local development

Use the task runner from the repository root:

```bash
task dev
```

Generate docs locally:

```bash
task docs
```

## Contributing notes

If you are exploring the project early, feedback on information architecture and workflow design is especially useful.
